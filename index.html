    SafeFeat documentation

    safefeat is an open source, MIT-licensed library for leakage-safe, point-in-time feature engineering from event logs. It builds ML features for each (entity_id, cutoff_time) using only events that occurred at or before each cutoff — so your model learns what it would actually know in production.
